This is a complete, ready-to-deploy project titled "The Hindsight Ledger." It uses the Streamlit framework (industry standard for data science prototypes) which allows you to build a web interface entirely in Python.

## How to Run (Instructions for your Project Report)

1. Install Python: Ensure you have Python 3.9+ installed.

2. Create Pyhon venv:
    python3 -m venv path/to/venv 
    source path/to/venv/bin/activate 

2. Install Libraries:
    Open your terminal/cmd in the project folder and
    run: pip3 install -r requirements.txt

4. Run the App:
    streamlit run app.py

5. The project will automatically open in your web browser.
